# CoH Sidekick Trademark Policy

The name **"CoH Sidekick"** and any associated logos or visual branding are unregistered trademarks of the CoH Sidekick project (Wednesdaywoe and contributors).

## What the AGPL license does and does not cover

The AGPL-3.0 license granted by this project applies to the **source code** of CoH Sidekick. It does not grant any license to use the project's **name, brand, or logos**.

This is deliberate and is standard practice for community-run open source projects (e.g., Firefox/Iceweasel, WordPress, Debian).

## Permitted uses

You may, without asking:

- Refer to "CoH Sidekick" by name in reviews, articles, tutorials, videos, bug reports, forum posts, and other commentary.
- State that your fork, tool, or service is "based on CoH Sidekick" or "compatible with CoH Sidekick data," provided it is clear that your project is **not** the official CoH Sidekick and is not endorsed by it.
- Link to the official CoH Sidekick repository and distribution channels.

## Uses that require permission

You may **not**, without prior written permission from the project maintainers:

- Publish, distribute, or host a fork, modified build, or derivative tool under the name "CoH Sidekick" or any confusingly similar name (e.g., "CoH Sidekick Pro", "CoH Sidekick Plus", "Sidekick for CoH", "CoHSidekick", etc.).
- Use the CoH Sidekick name or logos in a way that suggests your product, service, or organization is officially affiliated with, endorsed by, or sponsored by the CoH Sidekick project.
- Use the CoH Sidekick name or logos in advertising, promotional material, domain names, app store listings, or merchandise.

If you fork this code and publish your fork, please choose a distinct name for your fork. You are free to state that your fork is derived from CoH Sidekick.

## Why this policy exists

CoH Sidekick is offered freely to the City of Heroes community. This policy exists to protect the community from confusion — so that when someone downloads "CoH Sidekick," they know they are getting the genuine, freely-maintained project and not a rebranded commercial knockoff or a fork with different goals.

The code is yours to fork under AGPL. The name is how the community recognizes the official project — please respect that.

## Questions

If you are unsure whether your intended use is permitted, or if you would like to request permission for a use not covered above, please open an issue on the project's GitHub repository.
